# Android - Datos sin MVVM

Proyecto Android que implementa el consumo y manejo de datos sin utilizar el patron de arquitectura MVVM. Es un punto de comparacion para entender por que MVVM mejora la organizacion y testabilidad del codigo.

## Caracteristicas
- Acceso a datos directamente desde Activities
- Manejo de estado sin ViewModel
- Llamadas de red sin capa de abstraccion

## Leccion aprendida
Este enfoque muestra los problemas comunes de acoplamiento cuando no se separan las responsabilidades entre UI y logica de negocio. Compara este proyecto con su version refactorizada usando MVVM para ver la diferencia practica.

## Ejecutar el proyecto
1. Abre en Android Studio
2. Ejecuta en emulador o dispositivo fisico
